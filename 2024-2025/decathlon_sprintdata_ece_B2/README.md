# Decathlon Sprint Data (ECE B2)

Python + SQL workflow to scrape Decathlon product pages, clean fields, and load a dataset into MySQL.

## Main Files

- `data_collect.py`: Selenium scraper for listings and product detail pages.
- `database_init.py`: data cleaning and MySQL insert script.
- `decathlon_produits_details.csv`: collected dataset output.
- `sprintdata_decathlon.sql`: SQL schema/dump.
- `jupyter_database_init.ipynb`: notebook variant of the load pipeline.

## Requirements

- Python 3.x
- `pandas`
- `mysql-connector-python`
- `selenium`
- Edge WebDriver configured locally
- Local MySQL or MariaDB server

## Run Workflow

1. Update DB credentials in `database_init.py` (`host`, `user`, `password`, `database`).
2. Update Edge WebDriver path in `data_collect.py`.
3. Run scraper:
   - `python data_collect.py`
4. Load CSV into DB:
   - `python database_init.py`

## Outputs

- CSV dataset generated by the scraper.
- Rows loaded into the target MySQL tables.

## Notes

- Scraper selectors match the site structure from when the project was created.
- If page HTML changes, selectors may need to be updated.
