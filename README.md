# mini-projects

Monorepo of academic and personal projects, grouped by school year.

## Structure

- `2024-2025/`: ECE B2 projects (C, C++, Python, SQL).
- `2025-2026/`: personal projects (ML, finance, tooling, design docs).

## Project Index

### 2024-2025

- `2024-2025/Pacman_Allegro_ece_B2/`: Pacman clone in C with Allegro 4.
- `2024-2025/Ticket_reservation_cpp_ece_B2/`: console train ticket reservation system in C++.
- `2024-2025/decathlon_sprintdata_ece_B2/`: Decathlon product scraping and MySQL loading workflow.

### 2025-2026

#### ML Projects

- `2025-2026/MLProjects/notingAgentControlledByVoice/`: local voice agent (VAD + Whisper + local LLM + optional RAG + TTS).
- `2025-2026/MLProjects/pneumoniaCnnClassification/`: CNN pipeline for chest X-ray pneumonia classification (EDA, training, evaluation, Grad-CAM).

#### Finance Projects

- `2025-2026/financeProjects/finTech/IBAN_generator/`: bulk IBAN generator desktop prototype (Tkinter).
- `2025-2026/financeProjects/financialAnalysis/RouletteSimulator/`: notebook-based roulette simulation experiments.
- `2025-2026/financeProjects/financialModeling/k_means_realEstate_pricePredictor/`: custom K-means pipeline for real-estate price estimation.
- `2025-2026/financeProjects/financialSimulators/trading_simulator/`: Streamlit trading simulator with OHLCV data and PnL tracking.

#### Other Folders

- `2025-2026/schoolProjects/Gantt/`: project planning artifact (`.gan` file).
- `2025-2026/Docker/`: architecture diagrams for containerization ideas.
- `2025-2026/pythonProjects/`: reserved for Python-only projects (currently empty).

## Notes

- Most projects have their own `README.md` with setup and run instructions.
- Large datasets, models, virtual environments, and generated outputs should stay ignored via `.gitignore`.
